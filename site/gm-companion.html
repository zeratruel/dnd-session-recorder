What It Does

🎙️

Record Sessions

A Discord bot joins your voice channel and records each player separately. Perfect speaker attribution, no crosstalk issues.

📝

Transcribe Locally

Uses OpenAI's Whisper model running on your machine. No cloud, no API keys, no cost. Your audio never leaves your computer.

⚡

GPU Accelerated

Auto-detects NVIDIA GPUs for fast transcription. A 3-hour session transcribes in minutes. Works on CPU too — just slower.

🧹

Smart Condensing

Remove filler words, table talk, and off-topic chatter. Keep just the story, the roleplay, and the key moments.

🎭

Character Names

Map Discord usernames to character names. Transcripts read like a script — "DM: You enter the cavern..." not "User#1234".

🖥️

Control Panel

A web-based UI for managing everything. Transcribe, condense, configure — no command line needed after setup.

Get Started in 5 Minutes

1

Download

Clone or download from GitHub. Requires Node.js and Python.

2

Run Setup

Double-click setup.bat (Windows) or run ./setup.sh (Mac/Linux). It installs everything and walks you through configuration.

3

Create a Discord Bot

Go to the Discord Developer Portal, create an application, and paste your bot token when prompted.

4

Add to Your Server

Use the OAuth2 URL Generator to invite the bot with Connect, Speak, and message permissions.

5

Start Recording

Double-click start.bat, type !join in Discord, then !session start "Session Title". That's it.
